# CPEHUB Week 1 Projects

## Linear Algebra Solver
Solves a 2x2 system of equations using Cramer's Rule.

## Mini Calculator
Performs +, -, *, / operations with error handling for division by zero.

## Even/Odd Checker
Checks whether a number is even or odd using modulo operator.

## How to Run
Compile:
gcc filename.c -o output

Run:
./output
